# Go Food App
A mobile app which helps you find recommend restaurants near your place, built with React Native Expo.
<br/>
<br/>
日本語：お近くのおすすめレストランを検索するようにリアクトネイティブで開発されたモバイルアプリです。
   ![Logo](https://user-images.githubusercontent.com/35694395/85275989-5368b600-b4ab-11ea-8b97-828896d56b5c.png)



# To run this app

### 1. Clone into your own machine

### 2. Make sure npm (or yarn) and node have already been installed in your machine
 You can test by these command line:
 <br/>
 <br/>
 日本語：次のコマンドラインでテストできます：
 <br/>
 <br/>
   npm -v  
   node -v
   
### 3. Install all the needed node_modules package using:

  npm install
  
  or 
  
  yarn install
  

### 4. Run the app using:

npm start

or yarn start

### 5. Notes:
After running npm start (or yarn start), a new tab of the browser window will open (image below), you can scan QR CODE to open app test on real device mobile
<br/>
<br/>
日本語：npm start（またはyarn start）を実行した後、ブラウザーウィンドウの新しいタブが開きます（下の画像）。QRコードをスキャンして、実際のデバイスモバイルでアプリテストを開くことができます。
